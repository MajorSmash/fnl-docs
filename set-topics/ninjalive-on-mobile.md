SET TOPIC:** NINJALIVE ON MOBILE**
 welcome!
NinjaLive runs fluidsim in realtime - and with careful planning, it is running fine on Android and iOS.

https://www.youtube.com/watch?v=tOfyENY6Ww8&list=PLVCUepYV6TvOrOfQVLMCxl_JoU_cIkK8P&index=16&ab_channel=AndrasKetzer

https://drive.google.com/file/d/1I4dglPjeXLcNkSGxGok8sQCy59qgYcF9
(3) A video on VR experiments: https://twitter.com/_JasonCooper/status/1364968363328491529
(4) Once you have downloaded Live: compile Level 18, 19 on Android - already prepared for that
(5) Sidenote: Live 1.3 have been released recently, with serious mobile performance optimizations. Now, even an average mobile (eg Samsung A50) could be used to run ninja
