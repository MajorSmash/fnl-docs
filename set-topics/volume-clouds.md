SET TOPIC: **VOLUME CLOUDS** ☁️ 
 in the case of VolumeClouds, bounds and position are defined within the material (no mesh needed), using UVW coordinate offset and scaling. (1) Select "VolumetricCloud" Actor on Level, (2) Go to Actor Details Panel, and double-click on "Cloud Material" - so it will open up the actually used Cloud Material Instance. (3) In the Material, locate CloudU-Offset, *V-offset and *Altitude params.
