SET TOPIC: **INDIRECT DRIVE**
Direct Drive (Manual, Chapter 29.1 + https://youtu.be/vXalfRAnXak?t=3724) supports only a limited number of OutputMaterials *( 2 for Live 1.7 - 1.8 / 3 for Live 1.8.x.5 and 1.9 )* -- what if we'd like to drive more than 3 materials with the sim?
.
The solution is Indirect Drive: exposing ninja sim buffers and sim pos/scale params to ON DISK assets - that could be read by multiple materials
