SET TOPIC: **PULLING NEW NINJA VERSIONS FROM EPIC**
1. ninja supports all current UE version  *(UE 4.2 - 5.7)*
2. there are different ninja versions for different UE versions *(for compatibility reasons)*
3. **what ninja version are you working with?** --- Check **Edit /Project Settings /Project /Description**    (In the original, non-merged ninja project!)
4. what ninja version do you need for your specific UE version? Check *Manual Chapter 4* (https://discord.com/channels/850913821240983553/850925841793941565/855091427393273866), or the below list:
  - UE 4.20 - 4.22      Latest NinjaLive version: 1.0.1.0
  - UE 4.23 - 4.25      Latest NinjaLive version: 1.6.23.0
  - UE 4.26 - 4.27       Latest NinjaLive version: 1.8.4.5
  - UE 5.03                  Latest NinjaLive version: 1.8.5.5
  - UE 5.11 - 5.20        Latest NinjaLive version: 1.9.51.0
  - UE 5.32                   Latest NinjaLive version: 1.9.53.3
  - UE 5.44 - 5.6         Latest NinjaLive version: 1.9.54.4

5. How to get a specific ninja version? In EPIC launcher, users are required to set *their* local UE version, and the launcher automatically servers them with the matching ninja version
6. the process described under *point-5* ⬆️  could be messed up by **locally cached data** (of previous ninja versions) / and erasing previous local data might help --->  see these posts on removing local data before update: (A) https://discord.com/channels/850913821240983553/850913821827792940/1324479198403891211 , (B) https://discord.com/channels/850913821240983553/850925765997756468/1371965414522880111 , (C) https://discord.com/channels/850913821240983553/850925765997756468/1371968784570781797
