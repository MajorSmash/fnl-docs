SET TOPIC: **UPDATING NINJA**
1. The latest ninja version is always displayed at the manual front page (*currently 1.9.53.1*):      https://discord.com/channels/850913821240983553/850925841793941565/855091427393273866
2. You could check your local version number at: *Edit /Project Settings /Project /Description*
3. You could download the latest ninja from the marketplace - using EPIC Launcher /Vault
4.  once downloaded (to an empty folder), just overwrite your original ninja project (/Content/FluidNinjaLive) with the new data
     (using windows or mac file manager)
.
**Important**: in case you have modified original assets (eg. changed params in Material Instances) make sure you rename these before the override
