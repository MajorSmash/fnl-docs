🖐️ SET TOPIC: **NINJA vs NIAGARA**
- both ninja and niagara supports 2D fluidsim: that is fast in both cases
- niagara supports 3D sim as well: that is not very fast - but looks insane! / ninja can not do 3D sim
- ninja comes with large variety of tools/utilities, to drive 3D systems using 2D sim: Volumes use density as heightmap, Particle masses are driven by sim valocity, Foliage, Surfaces are using a combination of all buffers (including pressure) --- these tools are unique, and it would take a considerable amount of time to implement these for niagara: probably, that is the main advantage of using ninja
