SET TOPIC:  creating a cone like structure (tornado) with **NinjaTOOLS** <:NinjaTools:851394765647118346> 

1. create  a simple looping spiral and bake it to a flipbook
2. feed the baked flipbook to a Volumetric Material - eg. VolumeSmoke
The 2D grayscale flipbook-data will be used as a height-map to extrude a true 3D volume - while flipbook velocity could be used as a flowmap for procedural noise.
This video explains athe whole process: https://youtu.be/D0rAQR7CAIA
